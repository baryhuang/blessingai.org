Step 1: AI Script Writing

Use DeepSeek Chat or ChatGPT to quickly develop the script. Example prompt:

Please write a 3-minute video script with the theme "GOD IS SPEAKING". Requirements: 1. Set in modern urban life, showcasing miraculous moments throughout the day 2. Convey divine revelation and warmth through details of light and objects 3. Divide into six segments: opening(10s), morning(40s), commute(30s), workplace(50s), return journey(40s), ending(10s) 4. Highlight miraculous elements in each scene, ensuring continuity 5. Overall style should be authentic and emotionally engaging
Operation Steps Practical Examples
Input Theme Outline Theme: GOD IS SPEAKING
Segments: Opening 10s, Morning 40s, Commute 30s, Workplace 50s, Return 40s, Ending 10s
Style: Modern urban, Light miracles, Authentic emotion
Scene Description Opening: "At dawn, golden sunlight breaks through the city skyline..."
Morning: "Alarm rings, the first ray of sunlight casts supernatural shadows..."
Commute: "A white bird appears in the rearview mirror, roadside signs flutter in the wind..."

Step 2: AI Generate Storyboard

Midjourney prompt:
"City skyline at dawn, golden sunlight breaking through clouds, mysterious light shimmering between clouds, cinematic composition, high contrast, interplay of light and shadow --ar 16:9 --v 5 --style raw --q 2"
Operation Steps Practical Examples
Scene Description Morning: "Indoor morning scene, sunlight through curtains, creating mysterious light patterns on the wall, light flower effect --ar 16:9 --v 5"
Other Scenes Workplace: "Meeting room interior, focus on glowing note, colleagues giving meaningful glances, cinematic feel --ar 16:9 --v 5"

Step 3: Generate Video

Kling AI prompt:
"1. Start with aerial view of city skyline 2. Add dynamic sunlight penetration effect 3. Use slow motion for light shimmering 4. Transition to black screen with glowing subtitles"
  • Import Midjourney images sequentially
  • Set duration for each scene
  • Add smooth transition effects

Step 4: Voice-over Production

Hailuo AI prompt:
"Generate a whisper voiceover with requirements: 1. Use warm, magnetic young male voice 2. Speaking speed 0.9, solemn and mysterious tone 3. Genuine emotion with revelatory quality 4. Add echo effect at 'You, are being seen' 5. Include subtle morning breeze sounds"
Operation Settings
Voice Selection Young male, magnetic warmth, solemn tone
Audio Adjustments Speed 0.9, echo effects, morning breeze ambiance

Step 5: Add Background Music

Udio prompt:
"Create a soundtrack with requirements: 1. Opening using deep mysterious strings 2. Transition to warm piano melody 3. Add orchestral elements at climax 4. Return to solo piano for ending fadeout 5. Sync with video rhythm, emphasize key moments"
  • Select piano melody with string accompaniment
  • Add ambient sounds: city background, church bells
  • Adjust transition gradients

Step 6: Post-production Editing

Use CapCut to integrate all materials. Example instructions:

Import video, voiceover and music assets, trim to appropriate duration, add fade transitions, insert synchronized subtitles for key shots, ensure smooth pacing and emotional delivery.
  • Import video/audio assets
  • Adjust clip durations and transitions
  • Balance audio levels and add titles
